# Daftar Tugas Mandiri (Homework) untuk SEO Off-Page & Otoritas MATAPADI

Secara teknis (kode, kecepatan, *markup*, *security*), situs web MATAPADI sudah mencapai skor 100%. Namun, untuk mendominasi peringkat **Halaman Pertama Google** dan diutamakan oleh **AI Generative Engines (Perplexity, ChatGPT)**, Anda *WAJIB* mengeksekusi langkah-langkah bisnis/non-teknis berikut setelah web ini diluncurkan (*live*).

---

## 1. Dapatkan Backlink Berotoritas (Tugas Paling Kritis!)
Mesin pencari mengukur seberapa kredibel web Anda dari "siapa yang membicarakan Anda".
- [ ] **Jurnal/Kampus:** Usahakan mendapatkan tautan dari *website* universitas (berakhiran `.ac.id`). Jika ada mahasiswa/dosen yang meneliti MATAPADI, minta mereka menautkan URL web Anda di publikasi mereka.
- [ ] **Media Nasional:** Rilis *Press Release* ke portal berita teknologi atau agrikultur (Liputan6, Kompas, detik) saat alat ini divalidasi berhasil di persawahan skala besar.
- [ ] **Blog/Forum Pertanian:** Buat diskusi di forum agrikultur dan tautkan web Anda secara natural sebagai solusi hama burung.

## 2. Daftarkan ke Google Search Console
Mesin Google tidak akan langsung tahu web Anda eksis jika tidak diberi tahu.
- [ ] Buka [Google Search Console](https://search.google.com/search-console).
- [ ] Verifikasi kepemilikan domain (misal: `matapadi.com`).
- [ ] Masukkan URL sitemap Anda: `https://domainanda.com/sitemap-index.xml`.

## 3. Eksekusi Google My Business (Local SEO)
Jika konsumen mencari "Alat pengusir burung hama", Google akan memprioritaskan bisnis fisik yang terdaftar resmi.
- [ ] Buat akun *Google Business Profile*.
- [ ] Masukkan alamat bengkel/kantor riset MATAPADI secara lengkap.
- [ ] Tambahkan foto-foto implementasi alat (Raspberry Pi/Solar Panel) di lapangan.
- [ ] Minta testimoni Bintang 5 dari petani pertama yang menggunakan alat Anda.

## 4. Publikasi Data Riset Rutin (Strategi GEO & AEO)
Mesin AI seperti Perplexity tidak mencari "kata-kata promosi iklan", mereka mencari **"angka kuantitatif"**.
- [ ] **Rutin Rilis Studi Kasus:** Buat file `.md` baru di `src/content/research/` setiap 1-2 bulan.
- [ ] **Fokus pada Angka:** Gunakan judul seperti *"Efisiensi Panel Surya 50W pada Alat MATAPADI di Musim Hujan"*.
- [ ] Semakin banyak data faktual (GEO) yang ada di web Anda, semakin besar kemungkinan algoritma AI menjadikan Anda "Sumber Kebenaran Tunggal" (Single Source of Truth) untuk pengusir hama AI.
